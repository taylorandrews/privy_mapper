What is Reveal Estate?

Reveal Estate is a capstone project for the Galvanize Data Science Inmmersive. It uses scraped MLS data from a local company to cluster similar houses into relevant neighborhoods.

Learn more

How Does it Work?

Based on the data, houses are grouped using a machine learning algorithm. By learning from a set of manually clustered houses, this model can be applied to new cities.
